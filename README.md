# HANGMAN-0
Jeu Hangman (Bonhomme pendu) écrit en Pascal (Turbo Pascal ou Free Pascal)

Le jeu contient trois niveau : 

HANGMAN /LEVEL:BEGINNER - Débutant
HANGMAN /LEVEL:KNIGHT - Chevalier
HANGMAN /LEVEL:KING - Roi
